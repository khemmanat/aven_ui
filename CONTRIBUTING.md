# Contributing to AvenUI

Thanks for considering a contribution!

## How to add a new component

1. Create `lib/aven_ui/components/your_component.ex`
2. Follow the 1-file-1-module pattern
3. Add `@moduledoc` with at least one usage example
4. Add the component to `lib/aven_ui.ex` imports
5. Add it to `lib/aven_ui/mix/tasks/add.ex` component list
6. Open a PR with a description of what it does

## Reporting bugs

Open a GitHub issue with:

- AvenUI version
- Phoenix + LiveView version
- Minimal reproduction
